# **Jogo do numero secreto**

Jogo de adivinhação
O jogo consiste em chutar um numero dentro dos limites exibidos.

Caso o número do chute for correto, você ganha o jogo.

Após ganhar o jogo pode-se jogar novamente.

## Tecnologias:
- javaScript 
- HTML
- CSS
